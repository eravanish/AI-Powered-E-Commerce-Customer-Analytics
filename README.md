# 📊 AI-Powered E-Commerce Customer Analytics
An end-to-end data analytics project focused on transforming raw e-commerce transaction data into actionable business intelligence using Python, SQL, and Power BI.

---

## 📌 Project Overview
This project demonstrates how data filtering, machine learning, and advanced metrics can extract deep customer insights from e-commerce behaviors. It covers everything from programmatic data cleaning to relational database querying and interactive executive dashboards.

---

## 📂 Repository Structure

```text
├── dataset/             # Raw and processed e-commerce transaction files
├── python/              # Jupyter notebooks/scripts for data cleaning, preprocessing, and AI modeling
├── SQL/                 # Database schema scripts and analytical SQL queries
└── PowerBI/             # Power BI dashboard files (.pbix) and visualization reports
```

---

## 🛠️ Tech Stack & Skills Demonstrated
* **Data Preprocessing & AI (Python):** Cleaned complex datasets, handled missing variables, and implemented AI/statistical insights using libraries like `pandas`, `numpy`, and `scikit-learn`.
* **Database Management (SQL):** Structured relational tables, optimized complex joins, and aggregated metrics (e.g., Customer Lifetime Value, Retention Rate).
* **Business Intelligence (Power BI):** Designed interactive, executive-ready dashboard visualizations to track primary KPIs.
* **Version Control (Git/GitHub):** Managed codebase repository deployment, project structure, and change tracking.

---

## 🚀 Key Insights & Analytics Covered
1. **Customer Segmentation:** Grouping customers by purchasing patterns using AI/clustering techniques.
2. **Behavioral Analysis:** Evaluating purchase frequencies, cart abandonment patterns, and peak shopping hours.
3. **Revenue Optimization:** Tracking high-value customer tracks to maximize marketing ROI.

---

## 💻 Getting Started

### 1. Prerequisites
Ensure you have Python installed, a local SQL server configured, and Power BI Desktop active.

### 2. Installation & Setup
Clone the repository locally:
```bash
git clone https://github.com
cd AI-Powered-E-Commerce-Customer-Analytics
```

### 3. Execution Workflow
1. Navigate to `/python` to review the data cleansing scripts.
2. Load the cleaned data into your SQL client using scripts found in `/SQL`.
3. Open the `.pbix` report within `/PowerBI` to view the finalized analytics dashboard.
